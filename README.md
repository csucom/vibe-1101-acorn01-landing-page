# 📝 Simple To-Do Memo App

브라우저의 **LocalStorage**를 활용하여 데이터를 저장하는 심플하고 반응형인 할 일 관리 애플리케이션입니다. 
깔끔한 UI와 직관적인 기능을 제공하며, 별도의 백엔드 없이 브라우저만으로 동작합니다.

## ✨ 주요 기능

* **메모 추가**: 입력창에 할 일을 작성하고 '추가' 버튼이나 'Enter' 키를 눌러 목록에 추가할 수 있습니다.
* **실시간 목록 관리**: 추가된 항목은 즉시 리스트에 반영되며, 애니메이션 효과와 함께 나타납니다.
* **항목 삭제**: 각 메모 우측의 '삭제' 버튼을 클릭하여 개별 항목을 제거할 수 있습니다.
* **데이터 유지 (LocalStorage)**: 브라우저를 새로고침하거나 종료 후 다시 접속해도 작성한 메모가 그대로 유지됩니다.
* **반응형 디자인**: PC뿐만 아니라 모바일 기기에서도 최적화된 화면을 제공합니다.
